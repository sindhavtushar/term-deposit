About Project

Empowering banks with data-driven insights to enhance marketing strategies and customer engagement.

Customer Term Prediction

"Customer Term Prediction" is a machine learning project designed to predict whether a customer will subscribe to a term deposit based on their characteristics and interactions with the bank. By analyzing a bank's marketing dataset, we identify patterns in customer behavior to improve marketing effectiveness and decision-making.

Our approach involves data preprocessing, exploratory analysis, and building predictive models using advanced algorithms like, Decision Trees, Random Forests. We use tools such as Python, scikit-learn, and XGBoost to ensure accuracy and efficiency.

This project enables banks to enhance their marketing campaigns, optimize customer targeting, and boost term deposit subscription rates through intelligent predictions.

📊 Machine Learning Approach

📌 Data Preprocessing

The dataset consists of customer banking details and marketing interactions. The target variable (`y`) was converted to binary:

  • 1 → Customer subscribed
  • 0 → Customer did not subscribe

Feature Engineering:

  • 📈 Numerical Features: Scaled using StandardScaler
  • 🔤 Categorical Features: One-Hot Encoded

⚖️ Handling Class Imbalance

The dataset is imbalanced (88.3% not subscribed, 11.7% subscribed). To address this, we applied SMOTE (Synthetic Minority Over-sampling Technique).

🤖 Model Training

  • ✅ XGBoost (Weighted) – Best for balanced accuracy
  • ✅ Random Forest (Balanced) – Handles imbalance well
  • ✅ Gaussian Naïve Bayes – Works well with skewed data
  • ✅ KNN with SMOTE – Helps improve recall

🧑‍🤝‍🧑 Cross-Validation & Ensemble Learning

We used 5-fold cross-validation for model evaluation and combined the models into an Ensemble Model (Voting Classifier).

🏆 Final Model Performance

  • 📌 Ensemble Model Accuracy: 90.57%
  • 📊 Balanced Accuracy: 90.48% (Cross-validated)
  • 📈 ROC AUC:** 92.24% → Good class separation
  • 📉 PR AUC:** 59.65% → Handles class imbalance well
