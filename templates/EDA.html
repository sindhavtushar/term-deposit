About The Project

📌 Project Objective

This project aims to predict whether a bank customer will subscribe to a term deposit using machine learning. It analyzes banking marketing data to identify patterns in customer behavior.

📊 Dataset Overview

  • 📌 Total Records: 45,211
  • 📌 Target Variable: `y` (Subscribed or Not Subscribed)
  • 📌 Key Features:
    • 📞 `duration` (Call duration) – Most influential feature.
    • 📅 `month_sin` & `month_cos` (Seasonality effects).
    • 💰 `balance` (Customer bank balance).
    • 📈 `campaign` (Number of contacts during the campaign).
  • ⚠️ Class Imbalance: 88.3% not subscribed, 11.7% subscribed.

🔍 Exploratory Data Analysis (EDA)

The dataset was analyzed for missing values, outliers, and feature importance. Key findings:

  • ✔ No missing values found.
  • ⚠ Outliers detected in `balance`, `pdays`, and `duration`.
  • 📈 Feature importance: `duration` contributes 33.1% to predictions.
